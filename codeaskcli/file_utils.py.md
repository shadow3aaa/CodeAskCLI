### 代码分析报告

#### 1. 功能概述
该文件是代码分析 CLI 工具的文件处理模块，提供核心文件操作功能，包括：计算文件哈希（校验文件唯一性）、读取文件内容、按模式匹配项目文件、保存分析结果到 JSON 和 Markdown 文件。主要应用于代码分析场景，支持自动化扫描项目文件并生成结构化分析报告。

#### 2. 依赖项
- **标准库模块**  
  `os` (路径处理) | `glob` (文件匹配) | `hashlib` (哈希计算) | `json` (数据序列化)  
- **类型标注**  
  `typing.List`, `typing.Optional`

#### 3. 代码结构分析
**关键函数说明**  
- `get_file_hash()`  
  用 MD5 算法计算文件哈希值，用于检测文件变更  
- `read_file()`  
  安全读取文件内容，自动处理编码错误  
- `find_matching_files()`  
  按通配符模式递归搜索文件（支持 `*.py` 等格式）  
- `save_analysis_results()`  
  核心输出功能：  
  1. 将分析数据保存为 `.codeaskdata` JSON 文件  
  2. 生成 `SUMMARY.md` 总结文档  
  3. 为每个被分析文件创建同名 `.md` 分析报告
